## How to Write Each Learning Doc

Use this format:

```md
# Topic Name

## Short Explanation

## Why It Matters

## Laravel Implementation

## Example Request

## Example Response

## Common Mistakes

## Interview Questions

## Quick Revision Notes
```

Keep the writing practical. These notes should help with both real implementation and interview answers.
