# Exercises

This folder contains the day-wise hands-on practice.

Each day should have its own folder:

```txt
exercises/
|-- Day01/
|-- Day02/
|-- ...
`-- Day30/
```

Use this format inside each day:

```md
# Day XX - Topic Name

## Goal

## Concepts Learned

## Laravel Implementation

## Endpoints Built

## Request Examples

## Response Examples

## Problems Faced

## Tests

## Interview Questions

## Summary
```

The purpose is to build knowledge and hands-on experience together.
